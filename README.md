# PromptShield

PromptShield is a full-stack AI security platform that protects Large Language Models (LLMs) from prompt injection, jailbreak attempts, sensitive data leakage, and other prompt-based attacks.

## Project Structure

```
PromptShield/
├── frontend/     # Next.js + Tailwind CSS
├── backend/      # Express.js + MongoDB
```

## Tech Stack

### Frontend
- Next.js 16
- TypeScript
- Tailwind CSS
- Framer Motion
- Shadcn UI

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Helmet
- Morgan

## Features

- Prompt Injection Detection
- Jailbreak Detection
- Threat Scoring
- Prompt Scanner
- Dashboard & Analytics
- Authentication
- API Integration

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```
